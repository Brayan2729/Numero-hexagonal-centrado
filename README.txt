El siguiente proyecto puede ser ejecutado, a traves de los siguientes pasos:
1. Clonar el repositorio de github alojado de manera publica en el link: https://github.com/Brayan2729/Numero-hexagonal-centrado.git
2. Luego de clonar el repositorio, se puede editar por medio de un entorno de desarrollo como visual studio code
3. Teniendo abierto el proyecto en el entorno de desarrollo, lo puede ejecutar y depurar sin contratiempos
